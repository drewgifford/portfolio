# Portfolio

My personal portfolio/CV site. Now featuring cat pictures

Built with SvelteKit, Svelte 5, Tailwind CSS v4, and TypeScript. Uses Space Mono as the primary font and Iconify for icons.

## Development

```sh
bun install
bun run dev
```

## Stack

- [SvelteKit](https://kit.svelte.dev)
- [Tailwind CSS v4](https://tailwindcss.com)
- [TypeScript](https://www.typescriptlang.org/)
